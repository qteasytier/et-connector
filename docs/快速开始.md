# 快速开始

## 1. 下载和安装

### 1.1 获取安装包
从 [Gitee Releases](https://gitee.com/viagrahuang/et-connector/releases) 下载最新版本的安装包。

### 1.2 安装步骤
1. 运行安装程序
2. 选择安装目录
3. 点击"安装"按钮
4. 安装完成后点击"完成"

## 2. 首次运行

### 2.1 启动程序
- 双击桌面图标或从开始菜单启动"EasyTier 控制台连接器"
- 程序会在系统托盘显示图标

### 2.2 配置连接密钥
1. 右键点击系统托盘图标
2. 选择"设置"菜单项
3. 在弹出的对话框中输入 EasyTier 连接密钥
4. 点击"确定"保存设置

### 2.3 启动连接
1. 右键点击系统托盘图标
2. 选择"启动连接"菜单项
3. 程序会自动启动 EasyTier Core 服务

## 3. 使用 Web 控制台

- 双击系统托盘图标打开 Web 控制台
- 或在右键菜单中选择"打开 Web 控制台"

## 4. 开机自启

- 右键点击系统托盘图标
- 选择"开机启动托盘"菜单项（打勾表示已启用）
- 程序将在系统启动时自动运行

## 5. 停止和卸载

### 5.1 停止连接
1. 右键点击系统托盘图标
2. 选择"停止连接"菜单项

### 5.2 卸载程序
1. 从控制面板的"程序和功能"中找到"EasyTier 控制台连接器"
2. 选择卸载
3. 按照提示完成卸载过程

## 6. 常用快捷操作

| 操作 | 说明 |
|------|------|
| 双击托盘图标 | 打开 Web 控制台 |
| 右键菜单 → 启动连接 | 启动 EasyTier 服务 |
| 右键菜单 → 停止连接 | 停止 EasyTier 服务 |
| 右键菜单 → 设置 | 修改连接密钥 |
| 右键菜单 → 开机启动托盘 | 设置开机自启 |
| 右键菜单 → 退出软件 | 关闭程序 |
