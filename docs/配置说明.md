# 配置说明

## 1. 配置文件位置

配置文件位于 `%LOCALAPPDATA%/EasyTier/config.json`，程序启动时会自动创建。

## 2. 配置项详解

### 2.1 基本配置

```json
{
  "connectionKey": "your-connection-key",
  "autoStart": false,
  "autoReconnect": false
}
```

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `connectionKey` | string | "" | EasyTier 连接地址密钥 |
| `autoStart` | boolean | false | 开机自启开关 |
| `autoReconnect` | boolean | false | 自动回连开关 |

### 2.2 高级配置

```json
{
  "connectionKey": "your-connection-key",
  "autoStart": false,
  "autoReconnect": false,
  "serviceTimeout": 30000,
  "heartbeatInterval": 2000,
  "maxReconnectAttempts": 3,
  "reconnectInterval": 5000
}
```

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `serviceTimeout` | number | 30000 | 服务启动超时时间（毫秒） |
| `heartbeatInterval` | number | 2000 | 心跳检测间隔（毫秒） |
| `maxReconnectAttempts` | number | 3 | 最大重连次数 |
| `reconnectInterval` | number | 5000 | 重连间隔时间（毫秒） |

## 3. 配置修改方式

### 3.1 通过设置界面
1. 右键点击系统托盘图标
2. 选择"设置"菜单项
3. 在弹出的对话框中修改配置
4. 点击"确定"保存

### 3.2 手动修改配置文件
1. 关闭 EasyTier 控制台连接器
2. 找到配置文件：`%LOCALAPPDATA%/EasyTier/config.json`
3. 使用文本编辑器修改配置
4. 保存文件并重新启动程序

### 3.3 命令行参数
```bash
# 启动时指定连接密钥
EasyTierConnector.exe --connection-key "your-key"

# 启动时指定配置文件路径
EasyTierConnector.exe --config "C:\path\to\config.json"
```

## 4. 配置优先级

1. 命令行参数（最高优先级）
2. 配置文件
3. 默认值

## 5. 配置验证

程序启动时会验证配置的有效性：
- 检查 `connectionKey` 是否为空
- 验证 `serviceTimeout` 是否在合理范围内（1000-60000）
- 检查 `heartbeatInterval` 是否大于 0

## 6. 配置备份和恢复

### 6.1 备份配置
```bash
# 复制配置文件到备份位置
copy "%LOCALAPPDATA%\EasyTier\config.json" "C:\Backup\EasyTier\config.json"
```

### 6.2 恢复配置
```bash
# 从备份位置复制配置文件
copy "C:\Backup\EasyTier\config.json" "%LOCALAPPDATA%\EasyTier\config.json"
```

## 8. 配置迁移

从旧版本迁移配置：
1. 程序会自动从注册表 `HKEY_CURRENT_USER\Software\EasyTier\Connector` 迁移配置
2. 迁移完成后会删除旧配置
3. 新配置保存在 `%LOCALAPPDATA%/EasyTier/config.json`
